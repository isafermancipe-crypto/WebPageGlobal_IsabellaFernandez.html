    AI Development Record

    This section documents the AI conversations and tools used to build this webpage, as required by the assignment.

    AI Conversations Used
    Claude.ai — Main Development Conversation Claude.ai — C9 Data Fitting Logic

    Tool used: Claude (Anthropic) — claude.ai


    Prompts used: The AI was asked to create a complete single-page web application implementing C8 simulation mode with 5 probability distributions (Uniform, Triangular, Linear, Piecewise, Normal), interactive PDF graphs, probability calculations with visual shading, and C9 data modeling mode with CSV upload, parameter estimation, histogram overlay and interpretation.


    What was verified: Mathematical formulas for each PDF, normalization conditions, probability integral calculations (including numerical integration via Simpson's rule), parameter estimation algorithms (MLE for Normal, method of moments for others), and histogram binning logic.


    What was modified: Visual design, color scheme, layout, and interactive behavior were customized. The piecewise distribution editor and CSV parsing were refined for robustness.


    Testing: Each distribution was tested with known inputs and verified against theoretical values. For example, P(X ≤ b) for Uniform(0,1) with b=0.5 returns 0.5; Normal(0,1) with b=1.96 returns ≈0.975.
